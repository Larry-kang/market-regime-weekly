# 台灣市場週報

公開版的每週市場階段報告網站，面向台灣中文讀者。

## 目前內容
- 每週市場週報歸檔
- BTC / TAIEX / S&P 500 / Nasdaq 100 / 黃金 / 美國 10Y / DXY / VIX 標的頁
- 去識別化與公開範圍說明

## 站點工具
- 靜態站框架：MkDocs Material
- 發佈方式：GitHub Pages + GitHub Actions

## 本地預覽
```bash
uvx --from mkdocs mkdocs serve
```

## 建置
```bash
uvx --from mkdocs-material mkdocs build --strict
```
