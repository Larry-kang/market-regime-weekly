# 台灣市場週報

> 每週自動生成的公開市場觀察，面向台灣中文讀者。

## 這裡會看到什麼
- **最新週報**：每週一次的市場階段總結
- **標的頁**：BTC、TAIEX、美股、黃金、美債、美元指數、VIX
- **公開版原則**：只保留可公開的市場觀察，不放個人資金與私有策略
